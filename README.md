# BNB King Helper

An advanced tactical interface for the BNB Kingdom ecosystem. This tool provides real-time analytics, strategic advice, and streamlined management for your kingdom.

## Features

- **Real-time Analytics**: Monitor BNB price, contract liquidity, and your kingdom's production rates.
- **Strategic Advisor**: AI-driven recommendations for the most efficient next move (Buy vs Upgrade vs Save).
- **Battle Arena**: Optimized interface for battles with win/loss projections.
- **Modern UI**: Clean, dark-themed interface built for performance and clarity.

## Tech Stack

- React 18+
- TypeScript
- Tailwind CSS
- Ethers.js v6
- Vite

## Setup

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run development server:

   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
